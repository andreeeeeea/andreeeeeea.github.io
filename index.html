Baldur's Gate 3 Auto-Bot

Issue No. 2

For my 6-month final year project, I set myself the challenge of diving deep into systems programming and reverse engineering using Baldur's Gate 3. I wanted to see if it is possible to implement a bot that could automate combat in a complex RPG environment.

I built a C++ framework that reads the game's memory in real time, reconstructs characters and enemies, and simulates inputs to automate combat. This involved extensive memory analysis to map out data structures and offsets, enabling features like live game state monitoring and automated decision-making.
